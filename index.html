🎯 GitHub Pages — gratuito e público

O GitHub Pages transforma seu HTML em um site público acessível por qualquer pessoa, com URL fixa e custo zero.

🔗 Sua URL: https://seuusuario.github.io/controle-impl/

✅ Grátis para sempre · Sem limite de acessos · Qualquer equipe acessa pelo link

📋 Passo a Passo

1

Acesse github.com e crie uma conta gratuita.

2

Clique em New repository. Nome: controle-impl. Marque como Public.

3

Clique em uploading an existing file e envie este arquivo HTML renomeado para index.html.

4

Vá em Settings → Pages → Source: Deploy from branch → main → / (root) → Save.

5

Aguarde ~1 min. Compartilhe a URL com sua equipe!

💾 Fluxo com JSON (dados persistidos)

  1. Acesse a página publicada → Importar JSON (último backup)
  2. Crie módulos, adicione parâmetros, registre testes
  3. Ao terminar → Exportar JSON → salvar
  4. Próxima sessão: importe o JSON e continue

💡 Salve os JSONs no Google Drive ou OneDrive para backup automático na nuvem.
